# Link — URL Shortener

A minimal URL shortener built using semantic HTML5, CSS, and Vanilla JavaScript.  
Includes click tracking, expiry management, and persistent browser storage.

### Key Features

-  **Generate Short Links** — Create 6-character random codes or custom aliases.
-  **Click Tracking** — Monitor engagement with real-time counters.
-  **Expiry Management** — Set links to expire after 1hr, 24hr, 7d, or 30days.
-  **One-Click Copy** — Copy shortened links instantly using the Clipboard API.
-  **Persistent Storage** — All data saved in browser localStorage.
-  **Minimal Design** — Simple, responsive, and distraction-free design.
